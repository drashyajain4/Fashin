# Fashin
# 30 Minutes Fashion Delivery

A modern eCommerce fashion website that delivers trendy fashion products within **30 minutes**.  
This platform provides a fast, smooth, and user-friendly shopping experience with quick delivery services.

---

# Features

-  Browse fashion products
-  Search and filter products
-  Add to cart and checkout
-  Wishlist functionality
-  User authentication
-  Secure online payments
-  Order tracking
-  30-minute delivery service
-  Fully responsive design

---

# Tech Stack

## Frontend
- HTML
- CSS
- JavaScript
- React.js

## Backend
- Node.js
- Express.js

## Database
- MongoDB

## Authentication
- JWT Authentication

## Payment Integration
- Razorpay / Stripe

---

##  Website Purpose

The purpose of this project is to create a fast and efficient online fashion delivery platform that provides customers with fashion products delivered within 30 minutes.
